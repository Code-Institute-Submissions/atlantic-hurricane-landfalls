Introduction

This data dashboard will seek to provide the user with a way to visualize the hurricanes that hit the US Mainland since 1950 which was the year that marked the introduction of named storms.It is looking to see whether the frequency and or the intensity of storms has increased since then. The code for this project can be found here on github.

Notes on the Dashboard

There is a category selector that you can use to seperate the hurricanes by category.You can also click on the graphs to highlight only a particular category but to see the changes reflected on the map if you do this, you have to make the corresponding change with the category selector.
